1. Run Assignment4.slnx.
2. Start the solution (F5).
3. On the server program, type in the Server TCP Port Number, Server UDP Port Number and the file path you want the server to look at.
4. On the client program, type in the IP, Server Port Number, Server UDP Port Number, Client UDP Port Number and the file path of the location you want the files to be stored

Type "/l" to list the downloadable content in the server file path

Type "/d <ServerIP>:<ClientUDPPort> <filename>" to download the desired file
